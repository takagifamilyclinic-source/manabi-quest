🎒 まなびクエスト × Claude 連携手順書

子どもの「にがて」に合わせてアプリを育てるためのフロー（忘れ防止メモ）

これは何？

アプリに記録された成績を Claude Code に渡すと、苦手の分析とアプリの改善（問題の調整・データ追加など）をしてもらえます。この循環を回すための手順書です。

📱 iPadで
きろくを書き出す親ページ
▶
💻 PCで
Claudeに渡す/manabi
▶
🤖 分析と改善承認してから実装
▶
🔄 iPadに反映開き直すだけ

📱 iPad側（きろくの書き出し）

1
プロフィール画面を開くアプリを起動した最初の画面（プレイヤーえらび）。
2
右下の ⚙ を長押し1秒ほど おさえると PIN 入力が出ます。PIN（4桁）を入れて親ページへ。
3
「きろくを 書き出す」をタップ下の枠に記録（JSON）が表示され、自動で全選択されます。そのままコピー。
4
PCへ送るメール・メモ共有など、いつもの方法でOK。

💻 PC側（Claudeに渡す）

5
Claude Code を起動して /manabiまなびクエスト開発モードで再開します。
6
記録を貼り付けて依頼この記録から苦手を分析して: （ここにJSONを貼り付け）
7
Claudeがやること苦手Topの分析→改善の提案。提案にOKを出してから実装・git push まで進みます（勝手には変えません）。

🔄 反映の確認

8
数分待って iPad でアプリを開き直す公開ページが自動更新され、アプリが新しい中身に入れ替わります（Service Worker が新バージョンを取得）。

😵 こまったとき

更新が来ない → アプリを完全に閉じて開き直す（それでもダメなら1日おいてもう一度）。
PINを忘れた → PCで Claude Code に「まなびクエストのPINをリセットして」と頼む。アプリの更新が必要なので、反映まで少し時間がかかります（データは消えません）。
記録が大きすぎて貼れない → テキストファイルに保存して「このファイルを読んで」と渡す。
まなびクエスト（https://takagifamilyclinic-source.github.io/manabi-quest/）／この手順書もアプリと一緒に更新されます。
