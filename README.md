# まなびクエスト

小学生向けの学習アプリ。**算数バトル × 東北モンスター図鑑**。家庭用・完全無料・iPad対応のPWA。

- こどもが問題を解くとバトルが進み、クリアで東北モチーフのオリジナルモンスター(全身クリーチャー)をゲット
- モンスターは横手・秋田・東北の名物モチーフ全20体。各モンスターに郷土の豆知識つき
- 連続日数(ストリーク)で毎日の学習習慣づくり
- 学習記録は**すべてiPadの中だけ**に保存。外部に送信しません(実名も使いません)

## iPadに入れる手順

1. Safariで公開URL(`https://takagifamilyclinic-source.github.io/manabi-quest/`)を開く
2. 共有ボタン → 「ホーム画面に追加」
3. 以後はホーム画面のアイコンから起動。オフラインでも動きます

## いまの収録内容(第1弾)

- **算数ドリル**(自動生成・無限):低学年=たし算/ひき算(くり上がり・くり下がり)/九九、中学年=大きな数のたし算ひき算・かけ算・わり算
- 間違えたらその場で「やさしい解き方ガイド」を表示(ペナルティなし)
- モンスター図鑑20体・連続日数・2人プロフィール切替(低学年/中学年)

## 開発メモ

- 素のHTML/CSS/JS + PWA。ビルド不要・npm依存ゼロ(Node標準のみ)
- テスト: `npm test`(node標準テストランナー)
- ローカル確認: `python -m http.server 8420` → `http://127.0.0.1:8420`
- 更新の配布: `sw.js` の `CACHE` バージョンを上げて git push
- モンスター画像はPollinations.ai の無料経路で生成(課金ゼロ)

設計書は `docs/specs/`、実装計画は `docs/plans/` にあります。
