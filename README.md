# まなびクエスト

小学生向けの学習アプリ。**算数バトル × 東北モンスター図鑑**。家庭用・完全無料・iPad対応のPWA。

- こどもが問題を解くとバトルが進み、クリアで東北モチーフのオリジナルモンスター(全身クリーチャー)をゲット
- モンスターは横手・秋田・東北の名物モチーフ全20体。各モンスターに郷土の豆知識つき
- 連続日数(ストリーク)で毎日の学習習慣づくり
- 学習記録は**すべてiPadの中だけ**に保存。外部に送信しません(実名も使いません)

## iPadに入れる手順

1. Safariで公開URL(`https://takagifamilyclinic-source.github.io/manabi-quest/`)を開く
2. 共有ボタン → 「ホーム画面に追加」
3. 以後はホーム画面のアイコンから起動。オフラインでも動きます

## いまの収録内容

### 教科えらび(➗さんすう / ✏️かんじ / 🗣️えいご)
バトルに出かけると教科を選べます。いずれも同じバトル形式(10問→クリアでモンスター捕獲)。

### 算数(第1弾・自動生成)
学年別カリキュラム:1年=たし算ひき算 / 2年=九九+2桁筆算 / 3年=わり算+3桁 / 4年=2桁でわるわり算+2桁×2桁+大きな数。間違えたらその場で「やさしい解き方ガイド」(ペナルティなし)。

### 漢字(第2弾)
- 学年別配当漢字 全642字(1年80/2年160/3年200/4年202)
- 出題:**読み・書き・意味・部首・筆順(画数)**の全5形式を全学年(1〜4年)で収録
- すべて4択。読み・意味は公的な学年別配当漢字表に照合して収録

### 苦手自動調整
分野ごとの正答率を記録し、苦手な分野を自動で多めに出題(算数・漢字共通)。

### 親ページ(隠しボタン+4桁PIN)
プロフィール画面右下の⚙を長押し→PIN。子ごとの学習履歴・苦手トップ5の確認、学習記録の書き出し(端末内のみ・PINは伏字)、PIN変更、リセット。

### Claude連携(手順書)
苦手の分析とアプリ改善を Claude Code に依頼するフロー。親ページから記録を書き出して → Claude に渡す → 改善を実装 → アプリが自動更新される手順書: `docs/claude-flow-guide.html`（公開URL: https://takagifamilyclinic-source.github.io/manabi-quest/docs/claude-flow-guide.html ）

### やる気の仕組み(第3弾)
- **レベル / XP**:正解やクリアで経験値がたまり、ホームにレベルとXPバーを表示(正解1問=10XP・クリア=+50XP)
- **ごほうびポイント帳**:正解やクリアでポイントがたまる(正解1問=1pt・クリア=+5pt)。親ページで「ごほうび」を登録し、子のポイントと引き換えに交換(消し込み)。子は残高と一覧を見るだけ
- **モンスター進化**:同じモンスターを3回つかまえると1段階進化。図鑑と結果画面で進化後の姿(全身画像20体)＋「しんか!」演出。進化後の名前・画像も収録
- 学年をまたいでも進捗を保つデータ移行(schema v4)を実装済み

### その他
モンスター図鑑20体・連続日数・4学年プロフィール切替。

### 漢字 意味・部首・画数の3・4年拡張(第4弾その1)
- 漢字3・4年(全402字)に部首・画数データを追加し、**全学年(1〜4年)で読み・書き・意味・部首・筆順(画数)の全5形式×全642字が出題可能に**
- 部首の出題文は「ぶしゅは どれ?」、画数の出題文は「何画?」で統一。すべて4択

### 称号・バッジ(第5弾)
- ストリーク・レベル・図鑑コンプ・しんか・バトル回数・さんすう/かんじ/りかの正解数から自動で導き出す**バッジ23種**(emoji付き)。保存はせず、そのつど過去のがんばりから算出(導出方式)
- プロフィール画面から🏅バッジちょうを開いて獲得状況を確認。獲得ずみバッジをタップすると「称号」として名前の横に装備できる(称号えらび)
- バトルクリア時に新しく獲得したバッジをその場で演出表示
- データはschema v5(bestStreak/title を追加、v4→v5移行を実装済み)

### えいご 240語・読み上げ(第6弾その1)
- 学年別英単語 全240語(1〜4年各60語)。単語・かな読み・意味を収録
- 出題:**英→日(いみ)・日→英(たんご)**の2形式を全学年(1〜4年)で収録。すべて4択
- 単語は音声読み上げボタンつき(端末のSpeech Synthesisを利用・追加課金なし)

### りか 約100問・まめ知識付き(第7弾)
- 学年別理科 3・4年(約100問)。すべて4択で、まちがえたときに解説として「まめ知識」を表示(他教科と共通)
- 出題は選択肢シャッフル形式。1・2年は対象外

## 開発メモ

- 素のHTML/CSS/JS + PWA。ビルド不要・npm依存ゼロ(Node標準のみ)
- テスト: `npm test`(node標準テストランナー)
- ローカル確認: `python -m http.server 8420` → `http://127.0.0.1:8420`
- 更新の配布: `sw.js` の `CACHE` バージョンを上げて git push
- モンスター画像はPollinations.ai の無料経路で生成(課金ゼロ)
